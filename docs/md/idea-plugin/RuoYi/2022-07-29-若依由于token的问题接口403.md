## 若依前后端分离由于token的问题接口403

一边有Token，一边没Token

#### 第一种

请求头：

```javascript
headers : {
	isToken : false
}
```

#### 第二种

修改SecurityConfig

```java
// 使用 permitAll() 方法所有人都能访问，包括带上 token 访问
.antMatchers("/admins/**").permitAll()

// 使用 anonymous() 所有人都能访问，但是带上 token 访问后会报错
.antMatchers("/admins/**").anonymous()
```
